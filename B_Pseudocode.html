Bubble Sort

Pseudocode


			procedure bubbleSort( A : list of sortable items ) defined as:
			 do
			   swapped := false
			   for each i in 0 to length(A) - 2 inclusive do:
				 if A[ i ] > A[ i + 1 ] then
				   swap( A[ i ], A[ i + 1 ] )
				   swapped := true
				 end if
			   end for
			 while swapped
			end procedure
